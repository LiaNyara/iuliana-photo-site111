# Iuliana Photography Website – Deploy su Vercel

1. Vai su https://vercel.com
2. Accedi con Google o GitHub
3. Clicca su "Add New Project"
4. Carica questa cartella (scompattata) oppure collegala da GitHub
5. Clicca "Deploy" per pubblicare il tuo sito

Il sito supporta più lingue: italiano, inglese, francese, spagnolo, rumeno.
